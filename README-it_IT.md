# Tabella Aggiornata

| TypeScript       | Termine di insieme | Esempio         |
|-------------------|------------------|------------------|
| number            | Numero           | let x: number = 5;
| string            | Stringa          | let name: string = "John";
| boolean           | Booleano         | let isTrue: boolean = true;
| array             | Array            | let arr: number[] = [1, 2, 3];
| tuple             | Tupla            | let tuple: [string, number] = ["John", 25];
| any               | Qualsiasi        | let unknown: any = "Could be anything";
| void              | Vuoto            | function log(): void { console.log("Logging"); }
| never             | Mai              | function throwError(message: string): never { throw new Error(message); }
| enum              | Enum             | enum Color { Red, Green, Blue }
| object            | Oggetto          | let obj: { name: string; age: number } = { name: "John", age: 30 };
